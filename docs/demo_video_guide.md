# Demo Video Guide

Target length: 5 to 8 minutes

## Suggested Script

1. Introduce the problem.
2. Explain who the users are.
3. Show the datasets and reference material.
4. Open the Streamlit app and explain the input form.
5. Run one sample case.
6. Show the resource match, report, route/risk note, and emergency contacts.
7. Explain where AI is used:
   - retrieval
   - ranking
   - report generation
8. Show one failure or escalation scenario.
9. End with limitations and future improvements.

## Recording Checklist

- screen recording of app run
- clear narration
- one successful case
- one escalation/fallback case
- final conclusion
