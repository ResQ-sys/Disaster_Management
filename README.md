# AI Disaster Relief Resource Matching Agent

An Agentic AI platform that intelligently coordinates disaster relief operations by matching affected people with the nearest available shelters, hospitals, rescue teams, food supplies, and transportation resources. The system leverages multi-agent collaboration, Retrieval-Augmented Generation (RAG), and cloud-native architecture to prioritize requests, optimize resource allocation, and support faster, data-driven disaster response.

Key Technologies: LangGraph, LangChain, ChromaDB, Python, FastAPI, Streamlit, AWS-inspired serverless architecture (Lambda, Step Functions, DynamoDB, S3), and Large Language Models (LLMs).
